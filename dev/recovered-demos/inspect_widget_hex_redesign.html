Key Design Changes
FontsInter + JetBrains Mono → IBM Plex Sans + Mono
BackgroundDark #050607 + gradient → Clean white
AccentBlue #60a5fa → Purple #473982
Bordersrgba alpha → Solid #ecedf2
Radius10px / 999px pills → 3-6px subtle
TabsPill group, filled active → Underline indicator
MetricsInline in toolbar → Labeled grid cells
SelectionWhite alpha → Purple-tinted row
Structural Improvements
Summary grid. Key metrics (nodes, active/wall time, failure count) get labeled cells instead of being buried in the toolbar. Scannable at a glance.
Semantic status colors. Green (#1a8754), red (#c4320a), teal (#0e7490) are tuned for light backgrounds — more accessible than the dark-mode neons.
Consistent row heights. Timeline rows, label rows, and grid lines all snap to the same 40px rhythm, keeping bars aligned across the panel split.
Error block contrast. Lighter red-tinted background with solid border instead of glowing dark panel — error text is more readable at small sizes.
